# DSA Visualizer

A modern, interactive React application for visualizing Data Structures and Algorithms problems.
Currently features **LeetCode 124: Binary Tree Maximum Path Sum**.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Start the Development Server**
    ```bash
    npm run dev
    ```

## Features

-   **Interactive Visualization**: Step-by-step walkthrough of the algorithm.
-   **Modern UI**: Built with a clean, dark-mode first design.
-   **Controls**: Play, Pause, Step Forward/Backward.
-   **Tech Stack**: React, Vite, Framer Motion, Vanilla CSS (CSS Variables).

## Problems Implemented

-   **Binary Tree Maximum Path Sum (Hard)**: Visualizes the DFS traversal and path sum calculation logic.
